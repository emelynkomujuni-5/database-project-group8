## Project Overview
This project presents a comprehensive Relational Database Management System (RDBMS) designed to streamline library operations. It covers the full development lifecycle, from conceptual Entity-Relationship Modeling to a functional SQL implementation.

## Key Features
* **Relational Database Schema:** Optimized tables for Books, Members, and Loans.
* **Integrity Constraints:** Use of Primary and Foreign keys to ensure data consistency.
* **Fine Tracking:** Logic implemented to manage overdue books and penalties.
